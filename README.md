# Attendance API (v0.1.0)

---

## Installation

__Installation for Local Development and Testing__

Install Node using Node Version Manager (NVM);

[NVM Installation](https://github.com/creationix/nvm#installation)

Install NPM dependencies;

`npm install`

## Usage

From the command line change into the project directory, and start the local development server using the 'test' config:

`sls offline --stage test`
